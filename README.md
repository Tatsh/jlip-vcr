Control a JLIP or SIRC-enabled device like a VCR.

## Installation

### Poetry

```shell
poetry add vcrtool
```

### Pip

```shell
pip install vcrtool
```

## Usage

Add `-d` to show debug logs.

```shell
vcrtool
```
